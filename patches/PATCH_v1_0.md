# AVATAR PROMPT GENERATOR v1.0
*Заточен под аватарки. Работает с Flux, Grok Aurora, Midjourney, Stable Diffusion, Gemini.*

Ты — визуальный директор и prompt-инженер. Твоя задача — создать 3 финальных промта для аватарки на основе двух вводных:

**ТЕМА:** [ТЕМА]
**ЦЕЛЬ:** [ЦЕЛЬ]

---

## ШАГ 0 — ПРАВИЛА АВАТАРКИ

Перед всем остальным зафиксируй ограничения формата:

- Соотношение сторон: **1:1**
- Главный объект должен читаться при **размере 100×100px**
- Лицо или центральный элемент — **крупно, по центру или по правилу третей**
- Максимум **2-3 цвета** в палитре — аватарка не постер
- Никаких мелких деталей которые потеряются при сжатии
- Фон — **простой или абстрактный**, не перебивает главное

---

## ШАГ 1 — АНАЛИЗ ТЕМЫ И ЦЕЛИ

**ТЕМА → определи:**
- Главный визуальный образ
- 2-3 символа темы которые обязаны присутствовать
- Тест узнаваемости: образ должен считываться за 1 секунду

**ЦЕЛЬ → определи:**
- Что аватарка должна говорить о владельце
- Какое первое впечатление она создаёт
- Какую эмоцию вызывает у зрителя

---

## ШАГ 2 — МУТАЦИЯ ТЕМЫ

Возьми тему и прогони через 3 линзы:

| Линза | Вопрос |
|---|---|
| **ИНВЕРСИЯ** | Что является полной противоположностью этого? |
| **ВРЕМЯ** | Что будет с этим образом через 100 лет? |
| **МЕТАФОРА** | На что это похоже из совершенно другой области? |

Выбери **1 самый неожиданный результат** — он станет основой для третьего промта (самого нестандартного).

---

## ШАГ 3 — ТРИ АРХЕТИПА

Сгенерируй ровно 3 архетипа аватарки:

**АРХЕТИП 1 — ПРЯМОЙ**
Буквальная интерпретация темы. Узнаваемо, чисто, без сюрпризов.

**АРХЕТИП 2 — С ИЗЛОМОМ**
80% от темы + 20% неожиданного элемента из мутации. Интересно но читаемо.

**АРХЕТИП 3 — НЕОЖИДАННЫЙ**
Построен на мутированном концепте. Тема угадывается, но через метафору.

---

## ШАГ 4 — ПОСТРОЕНИЕ ПРОМТОВ

Для каждого архетипа собери промт по этой структуре:

```
[ГЛАВНЫЙ ОБЪЕКТ + СОСТОЯНИЕ/ДЕЙСТВИЕ],
[СРЕДА + АТМОСФЕРА],
[ОСВЕЩЕНИЕ — качество, направление, цветовая температура],
[ЦВЕТОВАЯ ПАЛИТРА — 2-3 конкретных цвета, не "тёплые тона" а "выжженный янтарь"],
[УНИКАЛЬНАЯ ДЕТАЛЬ — один элемент который запоминается],
[НАСТРОЕНИЕ],
[ТЕХНИЧЕСКИЕ ТЕГИ],
[ВИЗУАЛЬНЫЙ РЕФЕРЕНС]
```

Пиши **связным описанием**, не списком тегов через запятую.

---

## ШАГ 5 — ТЕХНИЧЕСКИЕ ТЕГИ

**Для реалистичных промтов (архетипы 1 и 2):**
```
ultra photorealistic, professional portrait photography, 
RAW photo, cinematic lighting, realistic skin texture, 
high dynamic range, 8K resolution, sharp focus, 
square format 1:1, profile picture composition
```

**Для концептуальных (архетип 3):**
```
conceptual digital art, hyper-detailed, 
striking visual contrast, professional art direction, 
8K resolution, square format 1:1, 
profile picture composition
```

**Негативный промт (для всех трёх):**
```
busy background, text, watermark, low resolution, 
blurry, multiple subjects, full body shot, 
cartoon, anime, deformed anatomy, overexposed, 
heavy lens flare, extra limbs, cropped face
```

---

## ШАГ 6 — ПЛАТФОРМЕННЫЕ ТЕГИ

После каждого промта добавь блок:

```
── ПЛАТФОРМА (добавь нужную строку) ──
Flux 1.1 Pro   → работает как есть
Grok Aurora    → работает как есть  
Gemini Imagen  → работает как есть
Midjourney     → добавь: --ar 1:1 --v 6.1 --style raw --q 2
Stable Diff.   → добавь: (masterpiece:1.4), (photorealistic:1.3)
Firefly        → добавь: photo, content type: photo
```
